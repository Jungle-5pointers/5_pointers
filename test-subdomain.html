서브도메인 테스트

배포된 사이트들을 테스트해보세요:

방법 1: Host 헤더를 사용한 테스트

  • 기본 서브도메인 서버 (localhost:3001)
  • 브라우저 개발자 도구 → Network → localhost:3001 요청에 Host 헤더 추가: testsite.localhost:3001

방법 2: /etc/hosts 파일 수정 (권장)

다음 내용을 /etc/hosts 파일에 추가하세요:

127.0.0.1   testsite.localhost
127.0.0.1   23qwe.localhost
127.0.0.1   31as.localhost
127.0.0.1   12asfd.localhost
127.0.0.1   21qw.localhost
127.0.0.1   312weq.localhost

방법 3: 로컬 DNS 서버 (dnsmasq)

dnsmasq 설치 후 *.localhost 와일드카드 설정

테스트 사이트들

  • testsite.localhost:3001 (새로 생성한 테스트 사이트)
  • 23qwe.localhost:3001 (Elegant Wedding Invitation)
  • 31as.localhost:3001 (Untitled)
  • 12asfd.localhost:3001 (Elegant Wedding Invitation)

curl 명령어로 테스트

curl -H "Host: testsite.localhost:3001" http://localhost:3001
curl -H "Host: 23qwe.localhost:3001" http://localhost:3001
